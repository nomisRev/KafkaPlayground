CommitStrategy

sealed interface CommitStrategy(source)

The strategy to apply to the "Offset commit manager". Offsets can be committed to kafka using different strategies:

  • Every n acknowledged records

  • Every interval duration, commit all acknowledged records

  • Every n acknowledged records or every interval duration, whichever comes first.
