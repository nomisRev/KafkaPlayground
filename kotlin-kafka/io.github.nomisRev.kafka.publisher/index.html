TransactionalScope
Link copied to clipboard
interface TransactionalScope<Key, Value> : PublishScope<Key, Value>

The DSL, or receiver type, of KafkaPublisher.publishScope, it allows for publisher configured with ProducerConfig.TRANSACTIONAL_ID_CONFIG, and ProducerConfig.ENABLE_IDEMPOTENCE_CONFIG set to true to work transitionally. Only one publisher with the same ProducerConfig.TRANSACTIONAL_ID_CONFIG can run at a given time. An expired producer, still works but will throw ProducerFencedException if attempt to make a transaction again.
