data class ProducerSettings<K, V>(val bootstrapServers: String, val keyDeserializer: Serializer<K>, val valueDeserializer: Serializer<V>, val acks: Acks = Acks.One, val other: Properties? = null)(source)

A type-safe constructor for KafkaProducer settings. It forces you to specify the bootstrapServer, and serializers for K and V. These are the minimum requirements for constructing a valid KafkaProducer.
